Description

The purpose of this website is to showcase and elaborate on my skills and projects that cannot be fully captured in my resume. It also includes links to my social media profiles, providing recruiters with additional insights about me. Initially, I created this site to enhance my HTML programming skills, but it has evolved into a passion project that allows me to experiment and grow my abilities further.

GitHub

Hosting + Development

This website is hosted on GitHub Pages, enabling me to modify and publish content from anywhere. GitHub handles all hosting and deployment, allowing me to update the site effortlessly by simply pushing changes. However, this approach does have some limitations. Specifically, it prevents me from using Flask, which restricts my ability to implement any backend functionality or create user-specific interactions, such as distinguishing between admin and general users.
