Description

A Python-based Discord bot, built with the Discord API library, designed for entertainment. It includes basic features for playing music, managing a Minecraft server, and querying a library database to provide information on the TCG game, Magic: The Gathering.

Hosting + Development

Hosted on a recycled Linux machine, this bot includes everything it needs to run, from the bot key to the server manager, and is maintained through GitHub for easy updates and management.
