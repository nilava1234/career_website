Planning Process

This project is currently a work in progress. The goal is to create an affordable and energy-efficient machine, either by repurposing an old one through recycling or purchasing a low-cost system. The plan involves using RAID and Linux to set up a storage and SSH server capable of handling video processing, file transfers, and hosting websites and bots.

Intended Usecase

This server will serve as my personal Plex server, enabling me to stream movies from anywhere. It will also host my websites, and migrating to my own machine is crucial as it will allow me to utilize React and other more complex frameworks. Additionally, it will provide a reliable environment to run my code without performance or dependency concerns, and it should be capable of hosting the Discord bot mentioned earlier, along with many other projects.
